# 薬院習字 — Yakuin Shuji

6種の筆で書道の表現を楽しむモバイルファーストWebアプリ。

## 特徴

- **UI-less設計** — 書くことに集中できる没入型インターフェース
- **6種の筆** — 羊毛筆（太）、羊毛筆、長々鋒、兎毛筆、竹筆、鶏毛筆
- **5層墨レンダリング** — コアインク、縁濃淡、毛筆テクスチャ、にじみ、かすれ
- **3つのジェスチャー** — 書く（1本指）、捨てる（2本指）、遊ぶ（落款タップ）
- **CYBER SURVIVOR** — 書道とは真逆のネオンアーケードゲーム（隠しモード）

## 操作方法

| 操作 | 説明 |
|------|------|
| 1本指ドラッグ | 筆で書く |
| 2本指タッチ | 紙を捨てる（筆選択に戻る） |
| 落款（左下の印）タップ | ゲームモード切替 |
| 筆ボタン（右上） | 筆の変更 |

## 技術スタック

- Vanilla JavaScript (フレームワーク不使用)
- HTML5 Canvas (2D)
- PWA対応 (Service Worker + Manifest)
- モバイルファースト設計

## ローカル実行

```bash
python -m http.server 8080
# → http://localhost:8080
```
